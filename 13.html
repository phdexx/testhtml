Задание 13: Компьютерные сети и адресация в Интернете

Тема: Компьютерные сети. Адресация в Интернете.

Что нужно знать:

  • Адрес документа в Интернете (URL) состоит из протокола, доменного имени, каталога и имени файла.
  • Каждый компьютер в Интернете имеет собственный адрес (IP-адрес), который является 32-битным числом.
  • IP-адрес состоит из адреса сети и адреса узла, определяемых с помощью маски сети.
  • Для задач на IP-адреса можно использовать модуль ipaddress в Python.

Пример задания:

Для узла с IP-адресом 15.51.208.15 адрес сети равен 15.51.192.0. Найдите наименьшее возможное количество единиц в двоичной записи маски подсети.

Решение:


from ipaddress import *
for i in range(32): 
    net = ip_network("15.51.208.15/" + str(i))
    sub = str(net).split("/")
    if sub[0] == "15.51.192.0": # нужная сеть
        print(net.prefixlen)
        break

Ответ: 18.
