Задание 16: Рекурсия. Рекурсивные процедуры и функции

Тема: Рекурсия. Рекурсивные процедуры и функции.

Что проверяется:

  • Вычисление рекуррентных выражений.
  • Индуктивное определение объектов.
  • Умение строить информационные модели объектов систем и процессов в виде алгоритмов.

Что нужно знать:

  • Для задания рекурсивной функции необходимо определить условие окончания рекурсии и рекуррентную формулу.
  • Решение задач с рекурсией можно выполнять вручную, используя электронные таблицы или собственные программы.

Пример задания:

Алгоритм вычисления функции F(n) задан следующими соотношениями:

  • F(n) = 2n при n ≤ 5
  • F(n) = F(n–2) + 3(F(n/2) + n) если n > 5 и n чётно
  • F(n) = F(n–1) + F(n–2) + F(n–3) если n > 5 и n нечётно

Чему равно значение функции F(99) + F(100)?

Решение (программа на Python):


def F(n):
  if n <= 5:
    return 2*n
  if n % 2 == 0:
    return F(n-2) + 3*F(n//2) + n
  else:
    return F(n-1) + F(n-2) + F(n-3)

print(F(99) + F(100))

Ответ: 142139494.
