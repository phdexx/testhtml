Задание 17: Перебор последовательности целых чисел. Проверка делимости

Тема: Перебор последовательности целых чисел. Проверка делимости.

Что проверяется:

  • Умение создавать собственные программы для обработки целочисленной информации.
  • Основные конструкции языка программирования.
  • Умение создавать программы на языке программирования по их описанию.

Что нужно знать:

  • Перебор всех целых чисел на отрезке и подсчет количества чисел, удовлетворяющих определенному условию.
  • Проверка делимости числа на другое число.

Пример задания:

Рассматривается множество целых чисел, принадлежащих числовому отрезку [1016; 7937], которые делятся на 3 и не делятся на 7, 17, 19, 27. Найдите количество таких чисел и максимальное из них. В ответе запишите два целых числа: сначала количество, затем максимальное число.

Решение:


count = 0
maxGood = 0
for n in range(1016, 7938):
    if (n % 3 == 0) and (n % 7 != 0) and (n % 17 != 0) and (n % 19 != 0) and (n % 27 != 0):
        maxGood = n 
        count += 1
print(count, maxGood)

Ответ: 1568, 7935.
