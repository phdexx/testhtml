Задание 15: Основные понятия математической логики

Тема: Основные понятия математической логики.

Что проверяется:

  • Знание основных понятий и законов математической логики.
  • Умение вычислять логическое значение сложного высказывания по известным значениям элементарных высказываний.

Что нужно знать:

  • Условные обозначения логических операций: отрицание (¬), логическое умножение (конъюнкция), логическое сложение (дизъюнкция), импликация (следование).
  • Таблицы истинности логических операций "И", "ИЛИ", "НЕ", "импликация".
  • Правила выполнения логических операций.
  • Формулы де Моргана для упрощения логических выражений.

Пример задания:

Укажите наибольшее целое значение A, при котором выражение ¬ДЕЛ(x,A) → (¬ДЕЛ(x,21) ∧ ¬ДЕЛ(x,35)) тождественно истинно (принимает значение 1) при любом натуральном значении переменной x.

Решение:


def Del(x, D):
    return x % D == 0

def f(x, A):
    return (not Del(x, A)) <= (Del(x, 21) <= (not Del(x, 35)))

for A in range(1, 1000):
    OK = True
    for x in range(1, 1000):
        if not f(x, A):
            OK = False
            break
    if OK:
        print(A)
        break

Ответ: 7.
