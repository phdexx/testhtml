Задание 8: Комбинаторика и системы счисления

Тема: Кодирование данных, комбинаторика, системы счисления.

Что проверяется:

  • Знание методов измерения количества информации.
  • Формализация понятия алгоритма.
  • Чтение и отладка программ на языке программирования.

Что нужно знать:

  • В русском языке 33 буквы: 10 гласных и 21 согласная.
  • Алфавит английского языка состоит из 26 букв.
  • Принципы работы с числами, записанными в позиционных системах счисления.
  • Формулы для расчета количества возможных слов.
  • Использование функций из модуля itertools в Python: combinations, permutations, product.

Пример задания:

Игорь составляет таблицу кодовых слов для передачи сообщений. В качестве кодовых слов он использует трёхбуквенные слова, в которых могут быть только буквы Ш, К, О, Л, А, причем буква К появляется ровно 1 раз. Сколько различных кодовых слов может использовать Игорь?

Решение (теоретическое):

Буква К может стоять на одном из трёх мест, остальные две буквы выбираются из оставшихся четырех букв: Ш, О, Л, А. Для каждой позиции буквы К существует 4^2 = 16 вариантов выбора остальных букв. Общее количество подходящих слов: 3 * 16 = 48.

Ответ: 48.

Решение (с помощью программы):


from itertools import product

def valid(word):
    return word.count('К') == 1

count = 0
for word in product('ШКОЛА', repeat=3):
    if valid(''.join(word)):
        count += 1
print(count)

Ответ: 48.
