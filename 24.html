Задание 24: Обработка символьных строк

Тема: Обработка символьных строк.

Что проверяется:

  • Умение создавать собственные программы для обработки символьной информации.
  • Умение строить информационные модели объектов систем и процессов в виде алгоритмов.

Что нужно знать:

  • Основы работы с текстовыми файлами в различных языках программирования.
  • Принципы поиска и обработки символов в строках.

Пример задания:

Текстовый файл 24.txt состоит не более чем из 10^6 символов X, Y и Z. Определите максимальное количество идущих подряд символов, среди которых каждые два соседних различны. Для выполнения этого задания следует написать программу.

Решение (программа на Python):


with open("24.txt", "r") as F:
    s = F.readline()

maxLen = curLen = 1
for i in range(1, len(s)):
    if s[i] != s[i-1]:
        curLen += 1
        if curLen > maxLen:
            maxLen = curLen
    else:
        curLen = 1

print(maxLen)

Ответ: 35 (примерное значение, зависит от содержимого файла).
